# Serverless Image API

## 📌 Project Overview
This project is a serverless image processing API built using:
- AWS Lambda
- API Gateway
- S3
- DynamoDB
- Terraform

## ⚙️ Features
- Upload image using API
- Store image in S3
- Process image using Lambda
- Store metadata in DynamoDB
- Retrieve image via API

## 🛠️ Tech Stack
- Python
- Terraform
- AWS Services

## 🚀 How to Run

1. Clone the repository
2. Configure AWS CLI
3. Run Terraform:

```bash
terraform init
terraform apply
